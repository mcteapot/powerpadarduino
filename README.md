
# NES Powerpad Arduino

User the NES Powerpad on Window, Mac, Linux or PS3 or any other Arduino based project you can think of!

If you make a game please sharte it with us [@cyborgdino](https://twitter.com/cyborgdino/)

## History

Since the release of the NES Powerpad only six games have ever been released for it. Now you create a game for this awesome peripheral from the 80's.
